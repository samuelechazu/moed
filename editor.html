Gemini API Key (Se guarda localmente)

Consigue una gratis en Google AI Studio. Se almacena localmente de forma segura en tu navegador.

¿Cómo conseguir tu Gemini API Key gratis?
  1. Ve a Google AI Studio (enlace abajo) e inicia sesión con tu cuenta de Google.
  2. Haz clic en el botón Get API Key en la esquina superior izquierda.
  3. Selecciona Create API Key y elige si deseas crearla en un proyecto nuevo o en uno existente.
  4. Copia tu clave generada (empieza con AIzaSy) y pégala en el campo de arriba.

Conecta tu editor directamente con tu repositorio de GitHub para poder cargar y publicar artículos en vivo en Vercel. Las credenciales se guardan de forma 100% segura únicamente en tu navegador.

GitHub Personal Access Token (PAT)

El token requiere permisos de lectura y escritura de contenidos (`contents: write`).

Repositorio de GitHub
¿Cómo crear tu Token (PAT)? Paso a Paso
  1. Ve a Settings → Developer Settings en GitHub.
  2. Selecciona Personal access tokens → Fine-grained tokens (Recomendado).
  3. Haz clic en Generate new token, asocia un nombre y selecciona tu repositorio en Repository access.
  4. En Repository permissions, busca Contents y actívalo como Read and Write.
  5. Genera el token, cópialo y pégalo arriba.
